# MCP环境安装
https://modelcontextprotocol.io/quickstart/server


```bash

{
  "mcpServers": {
    "hunyuanBingSearch": {
      "command": "uvx",
      "args": [
        "hunyuan-plugin-mcp-bingsearch"
      ],
      "env": {
        "BING_API_KEY": "$BING_API_KEY",
        "QUOTE_API_KEY":"$QUOTE_API_KEY"
      }
    }
  }
}

```
